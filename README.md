# 📖 Sobre

O **Biblioteca** é um projeto desenvolvido para colocar em prática conhecimentos em desenvolvimento **Full Stack**, utilizando **React** no frontend e **Node.js** no backend.

---

# 🚀 Objetivos

- 📘 Aprender React
- ⚙️ Desenvolver APIs REST com Node.js
- 🔐 Implementar autenticação
- 💾 Trabalhar com banco de dados
- 🔄 Desenvolver operações CRUD
- 🌐 Consumir APIs
- 📦 Organizar o projeto em camadas
- ☁️ Publicar a aplicação
- 🤝 Trabalhar em equipe

# ▶️ Como executar

### Clone o repositório

```bash
git clone https://github.com/SEU-USUARIO/biblioteca.git
```

## Frontend

```bash
cd frontend

npm install

npm start
```

## Backend

```bash
cd backend

npm install

npm run dev
```
